## 一、卤味制作

### 1. 香料配比

| 香料 | 用量 |
|------|------|
| 香叶 | 若干片 |
| 良姜 | 若干片 |
| 小茴香 | 一小撮 |
| 八角 | 七八个 |
| 肉蔻 | 一两个 |
| 桂皮 | 一两片 |
| 料酒 | 半酒杯 |

### 2. 食材准备

想吃什么就准备什么，基本什么都可以卤，干净新鲜即可。

### 3. 制作步骤

1. 在清洗食材前将香料用清水泡起来，可以用纱布包起来
2. 将食材清洗并焯水，处理干净
3. 将食材和香料一起入锅大火煮，入锅时加入三勺食盐，清水以淹没食材为宜
4. 水煮开后调至小火再煮 **35 分钟**
5. 停火时加入切好的藕片（可选，不卤藕片略过）
6. 闷 **10 分钟** 即可

### 4. 注意事项

- ⚠️ 不宜用高压锅煮，卤味太烂口感反而不好
- ⚠️ 不宜久煮，煮熟即可，筷子能轻松插穿肉即可

## 二、万能调料汁

### 1. 过油部分

| 材料 | 用量 |
|------|------|
| 姜末 | 多切一些 |
| 葱段 | 多切一些 |
| 干辣椒 | 少量（喜辣可多加） |
| 孜然粉 | 适量 |
| 花椒 | 适量 |

热油炸干姜末和葱段，然后放入干辣椒并**立即关火**（以不出现刺鼻辣味为准），关火后将热油浇至孜然粉和花椒上。

### 2. 不过油部分

| 材料 | 用量 | 说明 |
|------|------|------|
| 鸡精或十三香 | 少许 | 可选，根据个人口感 |
| 花椒油 | 少许 | 可选 |
| 清水 | 适量 | — |
| 精盐 | 一勺 | — |
| 香醋 | 少许 | 以吃不出酸味为宜 |
| 白醋 | 少许 | — |
| 香油 | 适量 | 以覆盖住佐料为宜 |
| 蒜泥 | 约半头大蒜 | **一定要蒜泥**，口感最佳，蒜末次之 |
| 青小米椒（或青美人椒） | 3～5根 | 喜辣多放，不喜少放，压制成泥或切碎末 |
| 糖色 | 少许 | 可选，上色用。**切忌用酱油上色**，会破坏口感 |
| 香菜或荆芥或薄荷 | 适量 | 可选 |
| 芝麻、花生碎 | 适量 | 可选 |
| 辣椒油 | 适量 | 可选 |

将以上佐料混合搅拌均匀即可。
