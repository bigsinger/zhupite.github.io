# 一、渡河问题（50题）

## 1. 两警两贼

### 题目
两位警察和两位小偷要过河，只有一艘小船，每次最多载两人。小偷不能与警察单独留在同一岸边，否则小偷会被逮捕。小偷和警察都能划船，船上必须有人才能划。

### 解法步骤
1. 两个小偷过河（G G →），一个小偷返回（← G）
2. 两个警察过河（P P →），一个警察返回（← P）
3. 两个小偷过河（G G →），一个小偷返回（← G）
4. 两个小偷过河（G G →），全部抵达对岸

## 2. 三警三贼

### 题目
三个警察押送三个小偷过河，船每次最多运两个人。若任意一边小偷人数 > 警察人数，警察就会被小偷杀掉。

### 解法步骤
1. 两个小偷过河（G G →），一个小偷返回（← G）
2. 两个小偷过河（G G →），一个小偷返回（← G）
3. 两个警察过河（P P →），一个警察+一个小偷返回（← P G）
4. 两个警察过河（P P →），一个小偷返回（← G）
5. 两个小偷过河（G G →），一个小偷返回（← G）
6. 最后两个小偷过河（G G →），全员安全

## 3. 猎人、狼、羊与白菜

### 题目
猎人带狼、羊、白菜渡河，每次只能带猎人和其中一项。猎人不在时，狼会吃羊，羊会吃白菜。

### 解法步骤
1. 猎人带羊过河，猎人返回
2. 猎人带狼过河，放下狼带羊返回
3. 猎人带白菜过河，猎人返回
4. 猎人带羊过河，全员安全

## 4. 猎人、猎狗与两只绵羊

2. 猎人带羊过河，放下羊带猎狗返回
3. 猎人带另一只羊过河，返回
4. 猎人带猎狗过河，全员安全

5. 两鬼过河（G G →），一鬼返回（← G）

## 8. 三只大老虎与三只小老虎

### 题目

### 解法步骤
1. a和b过河（a返回）
2. a和c过河（a返回）
3. A和B过河（B和b返回）
4. C和A过河（a返回）
5. a和b过河（a返回）
6. a和c过河，全员安全

## 天平称重类

### 1. 经典假币称重（3枚硬币）

**题目：** 有3个外观完全相同的硬币，其中1个是假币（假币比真币轻），用一架无砝码的天平，至少称几次能找出假币？

**答案：** 仅需1次。

**解法：**
1. 将3个硬币标记为A、B、C，分成三组（每组1枚）。
2. 用天平称A和B：若A端轻，则A是假币；若B端轻，则B是假币；若两端平衡，则C是假币。

### 2. 12枚硬币找假币

**题目：** 有12枚硬币，其中1枚是假币，假币和真币重量不同（未知轻重），用无砝码天平，至少称几次能确定假币并判断其轻重？

**答案：** 最多3次。

**解法：**
1. 将12枚硬币分成三组（每组4枚），标记为A组（A1-A4）、B组（B1-B4）、C组（C1-C4）。
2. 第一次称重：称A组和B组。
   - 若平衡：假币在C组，取C1、C2与A1、A2称第二次。
     - 若平衡：取C3与A1称第三次，平衡则C4是假币，不平衡则C3是假币（根据轻重判断假币属性）。
     - 若不平衡：记录轻重关系（如C1+C2重），取C1与A1称第三次，平衡则C2是假币，不平衡则C1是假币（对应第二次的轻重关系）。
   - 若不平衡（如A组重）：取A1、A2、B1与A3、B2、C1称第二次。
     - 若平衡：假币在A4或B3、B4中，取A4与C1称第三次。
     - 若A1+A2+B1重：假币在A1、A2（偏重）或B2（偏轻）中，取A1与A2称第三次。
     - 若A1+A2+B1轻：假币在B1（偏轻）或A3（偏重）中，取A3与C1称第三次。

## 计时类

### 1. 烧绳计时（测15分钟）

**题目：** 有两根粗细不均匀的绳子，每根烧完需1小时，如何用这两根绳子测量出15分钟？

**答案：** 通过两端点燃+二次点燃的方式，可精准测量15分钟。

**解法：**
1. 同时点燃第一根绳子的两端和第二根绳子的一端。
2. 当第一根绳子烧完时（耗时30分钟），立即点燃第二根绳子的另一端。
3. 从点燃第二根绳子另一端开始，到其完全烧完的时间即为15分钟。

**答案：** 通过"两端点燃+一端点燃"的组合，可测量45分钟。

**解法：**
1. 先点燃第一根香的两端和第二根香的一端。
2. 第一根香烧完时耗时30分钟，此时第二根香已燃烧30分钟（剩余30分钟燃烧时间）。

## 策略分配类

### 1. 海盗分宝

**题目：** 5个海盗抢得100枚金币，按规则由1号到5号依次提分配方案，方案需半数以上（含自己）同意才通过，否则提方案者被扔海里。所有海盗理性且优先保命、其次多拿金币，1号海盗如何提方案能存活且获利最多？

**答案：** 1号海盗分配方案：自己得97枚，2号0枚，3号1枚，4号0枚，5号2枚。

**解法（反向推理）：**
1. 只剩4号和5号时，4号提（100,0），自己同意即可通过，5号无金币。
2. 只剩3号、4号、5号时，3号提（99,0,1），争取5号支持（5号得1枚比后续0枚好），方案通过。
3. 只剩2号、3号、4号、5号时，2号提（99,0,1,0），争取4号支持，方案通过。
4. 1号需争取2票支持，给3号1枚（比2号方案多）、5号2枚（比2号方案多），即可获得3票（自己+3号+5号）。

### 2. 分蛋糕

**题目：** 两个小朋友分一块蛋糕，怎样分能让两人都觉得公平？

**答案：** "切者后选"原则。

**解法：**
1. 让其中一个小朋友切蛋糕，另一个小朋友先选择其中一块。
2. 切蛋糕的人会尽量切得均匀（避免自己拿到小的），先选的人能拿到满意份额，确保双方公平。

### 2. 杯子翻转

**题目：** 桌上7个杯口朝上的茶杯，每次翻转2个，能否经过若干次翻转让所有杯底朝上？

**答案：** 不能。

**解法：**
1. 7个杯口朝上变为朝下，每个杯子需翻转奇数次，总翻转次数为7个奇数相加（结果为奇数）。
2. 每次翻转2个杯子，无论翻转多少次，总翻转次数为2的倍数（结果为偶数）。
3. 奇数≠偶数，奇偶性矛盾，无法实现所有杯底朝上。
