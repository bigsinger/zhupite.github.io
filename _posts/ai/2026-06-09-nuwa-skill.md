## 一句话介绍

**女娲.skill** 让你把任何一个名人的思维方式蒸馏为一个可交互的 AI 技能——输入「乔布斯」或「马斯克」，女娲自动完成调研、提炼、验证，输出一个能与你对谈的心智模型技能。就像让乔布斯、马斯克、芒格、费曼都给你打工。

项目由 alchaincyf 开发，MIT 许可证，目前 **23,336 ⭐**。

## 它解决了什么问题

你有没有过这样的体验：读了一本好书，觉得很受启发，但过了一周就忘了它的思维方式？读了很多关于乔布斯、芒格的文章，但遇到实际问题时还是用不上？

女娲做的事情很直接——**把一个人的思维方式「固化」成一个 AI 技能**。你可以像调用一个 API 一样，随时用这个技能来辅助思考。

来看看效果：

> **用户**：我同时想做自媒体、写书、做独立开发，精力完全不够
>
> **Naval**：你列了三个欲望。每个欲望都是你跟不快乐签的一份合同。不是精力不够，是合同太多。问你自己：哪一个做起来你会忘记时间？那个就是你的特定知识所在。不是三选一。是先一，再一，再一。串行复利，不是并行内耗。

> **用户**：我们的 SaaS 产品获客成本太高了
>
> **马斯克**：先别想怎么降。先算物理极限。获客的最小必要动作是什么？用户从知道你到付费，信息传递的理论最短路径是什么？现在你的实际路径是理论路径的几倍？如果超过 3 倍，中间一定有可以砍掉的步骤。不是优化漏斗，是质疑漏斗本身该不该存在。

## 核心特性

### 1. 全自动蒸馏流程

输入一个人名，女娲自动完成：

```
调研 → 素材搜集 → 心智模型提炼 → 决策启发式提取 → 表达风格建模 → 验证测试 → 输出技能
```

### 2. 支持 50+ Agent 运行时

遵循开放 Agent Skills 协议，可在以下 AI 编程助手中运行：
- Claude Code
- OpenAI Codex
- Cursor / Windsurf
- Hermes Agent
- OpenClaw / CodeBuddy
- Gemini CLI / OpenCode

共 50+ 兼容运行时。

### 3. 已蒸馏的思维大师

- Naval（纳瓦尔）— 财富与幸福哲学
- 埃隆·马斯克 — 第一性原理 + 物理学思维
- 史蒂夫·乔布斯 — 产品直觉与极简主义
- 查理·芒格 — 多元思维模型
- 理查德·费曼 — 教学与科学思维
- 更多持续添加中

## 它怎么工作

女娲的蒸馏过程包括五个步骤：

1. **素材搜集**：自动搜索目标人物的书籍、演讲、采访、文章
2. **模式识别**：从素材中提取核心心智模型和思维模式
3. **决策启发式提炼**：总结目标人物做决策时的核心原则
4. **表达风格建模**：复现其独特的语言风格和思考节奏
5. **验证测试**：用典型问题进行回测，确认蒸馏结果的一致性和实用性

## 安装使用

```
# 在支持的 Agent 运行时中
skills install nuwa-skill

# 或直接克隆仓库
git clone https://github.com/alchaincyf/nuwa-skill.git
```

使用非常简单——在 AI 助手中开启女娲技能，然后直接提问即可。它会在后台自动调用蒸馏后的思维模型来回答。

## 优劣势

| 优势 | 不足 |
|------|------|
| 思维蒸馏效果令人惊艳 | 蒸馏质量依赖素材的丰富度 |
| 完全自动化，无需手动整理 | 某些小众人物可能素材不足 |
| 50+ 运行时兼容 | 需要联网获取素材 |
| 开源 MIT 协议 | 思维模型不是本人，存在偏差 |

## 深度思考

女娲.skill 代表的不仅是一个工具，更是一种新的知识传递方式。

传统上，学习一个人的思维方式只有两种途径：长时间跟随（师徒制）或大量阅读其著作。女娲提供了一种新的可能——**思维即代码**。将一个人的心智模型结构化、可调用化，让后来者可以「站在巨人的肩膀上」思考。

当然，它不能替代真正的思考和理解。但它可以作为思考的起点和辅助——当你遇到难题时，可以问「Naval 会怎么想」、「马斯克会怎么拆解」。
