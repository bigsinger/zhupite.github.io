## File Converter 是什么

**File Converter** 是一款极简的 Windows 文件格式转换工具，它巧妙地将转换功能集成在资源管理器的右键菜单中。选中一个或多个文件，右键选择"Convert"即可快速转换格式。无需打开任何复杂的界面，无需学习操作流程——转换就在你的指尖。它支持音频、视频、图片、文档等多种文件格式的转换和压缩。

项目在 GitHub 上已获得 **14,544** 颗星，自 2014 年开始开发，是 Windows 平台上最经典的右键转换工具。

> File Converter is a very simple tool which allows you to convert and compress files using the context menu in windows explorer.

## 核心特性

| 特性 | 说明 |
|------|------|
| 🖱️ 右键菜单 | 选中文件右键即可转换，无需打开应用 |
| 📦 批量处理 | 支持同时选择多个文件批量转换 |
| 🎬 音频转换 | MP3、WAV、FLAC、OGG、M4A 等格式互转 |
| 🎥 视频转换 | MP4、AVI、MKV、MOV、WebM 等格式互转 |
| 🖼️ 图片转换 | PNG、JPG、WebP、BMP、GIF 等格式互转 |
| 📄 文档转换 | PDF 压缩、Doc/Docx 格式互转 |
| 🗜️ 文件压缩 | 一键压缩视频、图片、PDF 文件体积 |
| ⚙️ 自定义配置 | 可自定义输出格式和质量参数 |
| 🔄 拖拽操作 | 支持将文件拖拽到转换器图标 |
| 🪟 Windows 原生 | 与资源管理器深度集成，体验原生 |

## 快速上手

**下载安装：**
1. 前往 [官网](https://file-converter.io/) 下载安装包
2. 运行安装程序，安装完成后资源管理器右键菜单即新增 "Convert" 选项
3. 首次使用可右键选择 "Convert" > "Settings" 配置默认格式

**基本使用：**
1. 在文件资源管理器中选中一个或多个文件
2. 右键点击，选择 "Convert" 菜单
3. 在下级菜单中选择目标格式（如 MP4→MP3、PNG→JPG）
4. 等待转换完成，转换后的文件在原文件同目录下生成

## 优劣势

**优势：**
- 右键菜单集成，使用极其便捷
- 无需打开独立应用程序
- 支持批量转换，效率高
- 支持音频、视频、图片、文档多种格式
- 支持文件压缩功能
- 完全免费开源
- 可自定义转换参数

**劣势：**
- 仅支持 Windows 平台
- 部分高级转换参数需要通过设置界面调整
- 大文件转换速度受限于系统性能
- 不支持 PSD/AI 等专业设计格式
- 转换格式数量与专业转换工具相比有限

## 适合谁用

- 想快速压缩媒体文件节省存储空间的人
- 任何追求高效简洁操作体验的 Windows 用户
