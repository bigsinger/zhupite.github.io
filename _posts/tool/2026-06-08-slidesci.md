## SlideSCI 是什么

**SlideSCI** 是一款专为科研人员设计的 PowerPoint 插件，解决日常科研汇报 PPT 制作中的各种痛点——从图片标题添加、元素位置对齐到代码块和 LaTeX 数学公式的插入。值得一提的是，这个插件 **99% 的代码由 AI 生成**，是 AI 辅助开发的绝佳范例。

开发者本人每周要做研究生科研进展工作汇报，深陷 PPT 排版之苦，于是借助 AI 开发了这套工具集。

> 不知道有没有人和我一样，对PPT积怨已久 😡

## 核心特性

| 特性 | 说明 |
|------|------|
| 📦 素材库 | 将 PPT 形状保存为素材，支持文件夹管理和搜索 |
| 🤖 AI 助手 | AI 辅助完成 PPT 制作任务 |
| 🏷️ 一键加图标题 | 图片像 Word 一样直接添加标题，告别手动对齐 |
| 📐 复制粘贴位置 | 将元素位置复制粘贴给另一个元素 |
| 🖼️ 一键图片对齐 | 多张图片自动多行多列整齐排列 |
| 📝 Markdown 插入 | 插入加粗、超链接、代码块等行内和块级样式 |
| 📐 LaTeX 公式 | 直接粘贴 LaTeX 数学公式格式到 PPT |
| 📷 便捷导出图片 | 快速将幻灯片内容导出为图片 |

## 快速上手

**安装要求：**
- Windows 系统
- Microsoft PowerPoint（建议使用较新版本）
- .NET 环境

**安装方式：**
1. 前往 GitHub Releases 页面下载最新版本
2. 运行安装程序，插件会自动集成到 PowerPoint 中
3. 重启 PowerPoint，在工具栏中找到 SlideSCI 选项卡

**使用示例：**
```bash
# 克隆项目
git clone https://github.com/Achuan-2/SlideSCI.git
# 使用 Visual Studio 打开并编译
```
