20|## you-get（Python命令行，强烈推荐）
21|
22|[you-get](https://github.com/soimort/you-get) 是一个老牌Python命令行下载器，Stars 56,800+，支持腾讯视频（v.qq.com）等国内外数十个网站。
23|
24|- **类型**: 命令行（CLI）
25|- **平台**: Windows / macOS / Linux
26|- **协议**: MIT（开源免费）
27|- **最新活动**: 2026年4月仍有推送
28|
29|### 快速上手
30|
31|```bash
32|# 安装
33|pip install you-get
34|
35|# 下载腾讯视频
36|you-get "https://v.qq.com/x/cover/xxx.html"
37|
38|# 查看可用画质
39|you-get -i "URL"
40|
41|# 指定画质下载
42|you-get -f "画质名" "URL"
43|```
