## Game Cheats Manager 是什么

**Game Cheats Manager (GCM)** 是一款开源的单机游戏修改器管理工具，帮助你一站式搜索、下载和管理各类单机游戏修改器。它聚合了 Fling、XiaoXing、Cheat Table 等多个来源的修改器，支持中英文搜索、一键下载和自动管理。你再也不需要在一个个可疑网站上翻找修改器了——GCM 将这一切整合到一个清爽的开源界面中。

项目在 GitHub 上已获得 **12,815** 颗星，深受单机游戏玩家喜爱。

> Easily download and manage single-player game cheats for your convenience.

## 核心特性

**多源聚合：**
| 来源 | 说明 |
|------|------|
| 🎮 Fling Trainers | 行业标准的修改器，质量可靠 |
| 🏯 XiaoXing Trainers | 专注于亚洲游戏的修改器，功能丰富 |
| 📊 Cheat Table | Cheat Engine 修改表，支持深度定制 |

**核心功能：**
- 🔍 **智能搜索**：支持中英文搜索，快速找到目标游戏修改器
- ⬇️ **一键下载**：点击即可下载，无需在网页间跳转
- 📂 **自动管理**：下载后自动分类管理，方便查找和使用
- 🆕 **版本更新**：自动检测修改器更新，保持最新状态
- 🎯 **热门推荐**：展示热门游戏修改器，发现新游戏
- 🌐 **多语言**：支持中文、英文等多语言界面

## 快速上手

**下载安装：**
1. 前往 [GitHub Releases](https://github.com/dyang886/Game-Cheats-Manager/releases) 下载最新版安装包
2. 安装后打开主界面
3. 在搜索框中输入游戏名称（支持中英文）
4. 一键搜索并下载对应的修改器
5. 下载完成后直接从 GCM 启动修改器

**使用场景示例：**
- 搜索 "Elden Ring" 即可找到法环的修改器
- 搜索 "怪物猎人 崛起" 找到对应的 XiaoXing 修改器
- 点击下载后自动保存到统一的管理目录

**官方网站：** <https://gamezonelabs.com>

## 优劣势

**优势：**
- 聚合多个修改器来源，覆盖面广
- 一键搜索下载，省去在多个网站间切换的麻烦
- 开源免费，代码透明安全
- 自动管理修改器文件，方便查找
- 支持更新检测，不再担心版本过期
- 干净无广告，用户体验好

**劣势：**
- 仅支持单机游戏修改器
- 修改器来源的可用性取决于第三方
- 部分游戏可能没有对应修改器
- 需要联网使用（搜索和下载功能）
