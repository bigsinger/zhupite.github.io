## mao-map 是什么

**mao-map**（毛主席足迹地图）是一个通过数据可视化技术，将毛泽东主席自 1893 年至 1976 年波澜壮阔的一生浓缩于一张可交互动态地图之上的开源项目。它让那段恢弘的历史「活」起来，清晰、生动地展现伟人在中国历史进程中的每一个关键抉择与地理印记。

项目同时提供 Web 在线版和基于 Tauri v2 构建的桌面客户端（支持 Windows 和 macOS）。

> 星星之火，可以燎原 / A single spark can start a prairie fire.

## 核心特性

| 特性 | 说明 |
|------|------|
| 🎬 时间轴播放 | 点击播放按钮按时间顺序观看 83 年人生轨迹动画 |
| ⏰ 时间定位 | 拖动底部时间轴快速定位到特定年份 |
| 🗺️ 交互式地图 | 多层级标记，智能显示不同样式的地图标记 |
| 📋 详情面板 | 点击标记查看完整历史事件列表 |
| 🔗 路径高亮 | 点击事件列表项高亮显示对应移动路径 |
| 📊 实时统计 | 动态显示移动次数、访问省市、国际移动等数据 |
| 📈 可视化图表 | 直观展示历史轨迹的统计信息 |
| 🖥️ 桌面客户端 | Tauri v2 构建，支持系统托盘、智能镜头跟随 |
| ⌨️ 快捷键支持 | 空格播放/暂停，方向键切换事件 |

## 优劣势

**优势：**
- 独特的主题视角，以地理轨迹串联历史
- Web + 桌面双端支持，覆盖更多使用场景
- 时间轴播放功能极具沉浸感和教育意义
- 桌面版支持系统托盘、开机启动等实用特性
- 开源社区协作，欢迎补充和完善数据

**劣势：**
- 数据完整性依赖社区贡献，可能存在遗漏
- 历史事件解读角度单一
- 桌面版仅支持 Windows 和 macOS
- 交互复杂度较高，移动端体验有待优化
