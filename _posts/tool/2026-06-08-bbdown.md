## 项目简介

BBDown 是一款专为 Bilibili（B站）设计的视频下载命令行工具，支持下载单视频、合集、番剧等多种内容类型，并提供 4K 画质和 Hi-Res 高品质音频选项。作为 nilaoda 开发的跨平台 CLI 工具，它曾是 B 站视频下载社区中最受欢迎的方案之一。

**注意**：该项目已归档停更，但依然是理解 B 站开放接口和视频下载技术的优秀参考实现。

## 核心功能

- **多种内容下载**：支持单视频、合集、UP 主全部视频、番剧、课程
- **画质选择**：最高 4K 分辨率，支持 HDR、杜比视界
- **音频下载**：Hi-Res 高品质音频、杜比全景声
- **弹幕/字幕/封面**：同步下载 XML/DANMAKU 格式弹幕、CC 字幕、视频封面
- **Cookie / TV 登录**：支持网页端和 TV 端登录方式，解锁更高画质
- **FFmpeg 合并**：自动调用 FFmpeg 合并视频和音频轨道
- **跨平台**：基于 .NET 运行时，支持 Windows / macOS / Linux

## 使用方式 / 安装

### 直接下载（Windows）

从 [Releases 页面](https://github.com/nilaoda/BBDown/releases) 下载对应平台的预编译二进制文件。

### dotnet 工具安装

```bash
dotnet tool install --global BBDown
```

### 基本使用

```bash
# 下载单个视频（最高画质）
BBDown https://www.bilibili.com/video/BV1xx411c7mD

# 下载番剧
BBDown https://www.bilibili.com/bangumi/play/ss12345

# 下载合集
BBDown https://www.bilibili.com/video/BVxxx --multi-download

# 指定画质
BBDown https://www.bilibili.com/video/BVxxx -q 4K
```

第一次使用需要设置 Cookie 或使用 TV 登录以获取更高画质权限。

## 技术原理

BBDown 的核心技术原理是模拟 Bilibili 客户端向 API 发起请求，获取视频流地址、弹幕数据、字幕信息等资源。它利用了 B 站开放的 API 接口（需要登录凭证），通过解析视频 CID 来获取对应画质的媒体流。

在下载流程上，BBDown 首先获取视频基本信息，然后根据用户指定的画质参数选择最佳的音频和视频流，最后通过 FFmpeg 进行封装合并。弹幕以 XML 或 DANMAKU 格式保存，支持后续在播放器中加载。

## 社区与生态

BBDown 虽然已归档，但其代码库仍然是许多社区衍生工具的基础。它简洁的 CLI 设计和完整的 B 站 API 封装，为后来者提供了宝贵的参考。一些基于 BBDown 改造的 GUI 封装、Web 服务版工具仍在社区中活跃。

对于仍在使用 BBDown 的用户，建议关注其后续的 API 兼容性问题——随着 B 站接口的持续调整，已停更工具可能会出现功能失效的情况。
