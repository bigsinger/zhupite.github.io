# OpenCalc：简洁优雅的开源 Android 计算器，30 万+ 下载量

## 一、核心特性

### 1.1 应用特性

| 特性 | 说明 |
|------|------|
| **极致轻量** | APK 仅约 **2MB** |
| **无多余权限** | 仅申请最小必要权限 |
| **锁屏可用** | 锁屏界面直接弹出计算器 |
| **快速磁贴** | 下拉通知栏一键启动计算 |
| **横竖屏适配** | 竖屏基础模式 / 横屏科学模式 |

### 1.2 主题系统

| 主题 | 效果 |
|------|------|
| **浅色主题** | 标准白色背景，清晰易读 |
| **深色主题** | 暗色背景，护眼省电 |
| **纯黑主题（AMOLED）** | 像素级纯黑，AMOLED 屏幕极致省电 |
| **Material You** | Android 12+ 动态取色，随壁纸色自动适配 |

### 1.3 科学模式

横屏自动切换的科学模式，完整支持：
- 三角函数（sin / cos / tan + 反函数）
- 对数（log / ln）
- 指数与幂运算（x² / x³ / xʸ / eˣ / 10ˣ）
- 平方根与 n 次方根
- 阶乘、π、e 常数
- 括号嵌套运算
- 角度/弧度切换

### 1.4 历史记录

- 完整的运算历史列表
- 支持点击历史条目重新加载表达式
- 可逐条删除或清空全部

## 二、自研代数引擎

OpenCalc v2.0.0 起使用了**自己的代数系统**，而非 mXparser 等第三方库。原因：

| 对比维度 | mXparser | OpenCalc 自研引擎 |
|----------|---------|-----------------|
| 库体积 | 约 6MB | **约 2MB（整体 APK）** |
| 精度 | 已知存在精度问题 | 自研可控 |
| 功能自由度 | 受限于第三方 API | 完全自定义 |
| 维护控制 | 依赖上游更新 | 自主维护 |
