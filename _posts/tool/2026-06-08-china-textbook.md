## ChinaTextbook 是什么

**ChinaTextbook** 是一个汇集了国内小学、初中、高中及大学各学科 PDF 教材的开源项目。项目创建初衷是为了让更多人免费获取正规教材资源，促进义务教育的普及，消除地区间的教育贫困，同时帮助海外华人的孩子继续了解国内教育。

该项目的 GitHub Stars 高达 **72,610**，是国内开源社区中极具影响力的教育类项目之一。

> 项目的由来：虽然国内教育网站已提供免费资源，但大多数普通人获取信息的途径依然受限。有些人利用这一点，在某站上销售这些带有私人水印的资源。为了应对这种情况，项目作者将这些资源集中并开源。

## 核心特性

| 特性 | 说明 |

| 📚 全学段覆盖 | 小学、初中、高中、大学各阶段教材 |

| 📄 PDF 格式 | 标准 PDF 格式，方便阅读和打印 |

## 快速上手

**在线查看或下载：**
1. 打开 GitHub 仓库：<https://github.com/TapXWorld/ChinaTextbook>
2. 按目录结构找到对应学段和科目
3. 点击 PDF 文件在线预览或下载到本地

**仓库目录结构示例：**
```
小学/
  数学/
    人教版/
      一年级上册.pdf
      一年级下册.pdf
      ...
初中/
  数学/
  语文/
  ...
高中/
  数学/
  语文/
  ...
```

## 优劣势

**优势：**
- 覆盖全学段、全学科，内容极其丰富
- 完全免费开源，拒绝教育资源商业化
- 72K+ Stars，社区认可度极高
- 助力教育公平，消除信息差
- 海外华人子女学习国内教材的便利渠道

**劣势：**
- 教材版本以人教版为主，其他版本覆盖有限
- PDF 直接存储在 GitHub 仓库，大量下载可能受网络限制
- 教材更新可能存在一定延迟
- 无许可证声明，使用需注意版权边界
