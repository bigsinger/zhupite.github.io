新装的 Windows 系统里有一些你可能用不上的预装应用，有些系统设置默认偏保守（比如隐私、性能），还有一些界面细节你可能想按自己的习惯调整。微软官方提供了不少设置入口，但分散在各处，改起来挺烦琐。

[**Winhance**](https://github.com/memstechtips/Winhance) 是一个开源的 Windows 优化工具，C# 开发，GitHub 10.9K Stars。它把这些散落的设置项集中到一个界面里，你可以一键调整——移除不需要的应用、优化系统性能、调整界面样式、创建自定义安装镜像。

---

## 一、四大功能模块

### 1. 软件与应用管理

**Windows 自带应用**——查看和移除系统预装应用（包括 Xbox、Cortana、OneDrive、Edge 等），也可以安装被移除的组件。界面带搜索和说明标签，每个应用都有简要解释，方便判断是否需要保留。

**外部应用批量安装**——通过 WinGet 一键安装常用软件，分类包括浏览器、多媒体、文档工具等。

### 2. 系统优化

所有优化项以开关形式呈现，方便预览和调整：

| 设置项 | 说明 |
|--------|------|
| **UAC 通知级别** | 控制用户账户控制的弹窗频率 |
| **隐私设置** | 关闭遥测、定位追踪、广告 ID、诊断数据等 |
| **游戏优化** | 禁用后台动画、优化 GPU 调度、启用游戏模式 |
| **Windows 更新** | 控制更新策略，推迟功能更新 |
| **电源方案** | 切换高性能/平衡/节能模式 |
| **声音设置** | 禁用系统音效、通信活动时静音等 |
| **通知偏好** | 关闭不必要的通知和弹窗 |

### 3. 界面定制

Windows 11 的一些界面改动不是每个人都喜欢。Winhance 提供了一键恢复或调整的选项：

- 深色/浅色主题切换
- 任务栏对齐方式（居中或左对齐）
- 任务栏图标和小组件开关
- 开始菜单布局调整
- 文件资源管理器优化（显示扩展名、隐藏文件夹、恢复经典右键菜单等）

### 4. 高级工具

**WIMUtil（Windows 安装镜像工具）**——创建自定义 Windows ISO 安装镜像。可以把当前系统驱动注入镜像，集成 Winhance 的设置项，这样新装系统后不用再手动调整一遍。

**自动应答文件生成**——根据你当前的 Winhance 设置生成 `autounattend.xml`，用于无人值守安装。

## 二、安装与使用

### 安装

支持 Windows 10/11（x64）。安装方式很简单，在 PowerShell 中执行：

```powershell
irm "https://get.winhance.net" | iex
```

也可以从 [GitHub Releases](https://github.com/memstechtips/Winhance/releases) 下载安装包。安装包同时包含可安装版和便携版。

### 使用界面

Winhance 的主界面按功能分区排列，左侧导航切换模块。每个设置项都有开关和文字说明——不是那种"看不懂在改什么"的工具。调整后可以保存为配置文件，在另一台机器或重装系统后导入，不用重复设置。

### 配置文件

Winhance 支持将当前设置导出为配置文件（`.json`），方便在多台机器间同步，或在新装系统后一键恢复之前的优化策略。

## 三、适用场景

| 场景 | 说明 |
|------|------|
| **新装系统** | 安装完 Windows 后，用 Winhance 一次性移除预装应用、调整隐私和性能设置 |
| **多台电脑维护** | 保存一份配置文件，在多台机器上导入相同的优化策略 |
| **重装系统** | 配合 autounattend.xml 和自定义 ISO，新系统装好就是优化后的状态 |
| **游戏优化** | 关闭后台进程、动画特效，最大化系统资源给游戏 |
| **隐私保护** | 一次性关闭遥测、定位、广告 ID 等数据收集选项 |

---

## 四、注意事项

- **建议先创建还原点**——虽然每个设置项都可逆，但批量修改系统设置前，建议先创建系统还原点或备份。
- **每个设置项都有说明**——界面中每个开关都附带了说明文字，不要盲目全关，看清楚每项的作用再操作。
- **部分选项需要重启生效**——一些系统级设置（如任务栏布局、部分性能优化）在更改后需要重启才能完全生效。
- **Windows 更新可能会覆盖**——系统大版本更新后，部分优化项可能需要重新应用。
