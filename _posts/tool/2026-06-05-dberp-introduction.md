## DBErp 是什么

DBErp 是一款面向**中小企业和个体经营者**的开源进销存（进货-销售-库存）管理软件。它帮助传统线下生意把进销存、财务、零售等核心业务流程搬到电脑上，**替代手工记帐和 Excel 表格**。

> 一句话：让小微企业也能用上专业的进销存管理系统，而且是免费的。

| 维度 | 说明 |

## 核心功能

### 📦 采购管理
- 采购订单录入、审核、入库全流程
- 供应商管理及历史采购价格查询
- 采购退货处理
- 采购报表统计（按供应商/商品/时间维度）

### 🛒 销售管理
- 销售出库单、销售退货
- 多价格策略（批发价/零售价/会员价）
- 客户管理及信用额度控制
- 销售报表与毛利分析

### 🏪 零售管理（POS）
- 收银台模式，支持扫码枪
- 小票打印
- 会员储值、积分兑换
- 多门店收银数据汇总

### 📊 库存管理
- 商品入库/出库/调拨/盘点
- 库存预警（低于安全库存自动提醒）
- 库存成本核算（移动平均法）
- 多仓库管理

### 💰 财务管理
- 应收/应付账款管理
- 收入/支出流水
- 资金账户管理
- 利润报表

### 📋 报表系统
- 库存状况表、出入库统计表
- 销售排行、采购分析
- 经营日报/月报
- 利润统计

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端语言 | PHP |
| 框架 | ThinkPHP / Laravel（视版本而定） |
| 前端 | jQuery + Bootstrap（传统版）/ Vue（新版） |
| 数据库 | MySQL / MariaDB |
| 部署 | 支持 Docker、宝塔面板一键部署、XAMPP/WAMP 传统部署 |

**技术特点**：
- 轻量级，对服务器要求低（1 核 1G 即可跑）
- 无需安装客户端，浏览器访问即可
- 支持局域网和公网部署
- 二次开发门槛低（PHP 上手容易）

## 不足

| 不足 | 说明 |
|------|------|
| ❌ **界面风格** | 传统管理后台风格，审美较为老派 |
| ❌ **生态系统** | 对比同类型开源 ERP 项目，社区活跃度和插件数偏低 |
| ❌ **移动端** | 对移动端适配有限，没有原生 App |
| ❌ **高级功能** | 强依赖付费插件（如电子发票对接、物流追踪等） |
| ❌ **多语言支持** | 主要面向中文用户 |

## 部署建议

**最低配置**：
- 服务器：1 核 CPU、1GB 内存
- 系统：Linux / Windows
- PHP：7.4+
- MySQL：5.7+

**推荐部署方式**：
1. 宝塔面板一键部署（最快）
2. Docker Compose 部署（推荐生产环境）
3. 传统 LAMP/WAMP 环境部署

## 总结

DBErp 是一款**功能实用、上手简单、部署轻量**的开源进销存系统。对于预算有限但需要数字化管理的中小企业和个体经营者来说，是一个低成本、高性价比的选择。

如果你的需求是：
- ✅ 传统贸易/批发/零售的进销存管理
- ✅ 替代手写帐本或 Excel 管理库存
- ✅ 需要本地部署、数据自主可控
- ✅ 预算有限但功能不能太差

那 DBErp 值得一试。如果你的业务链路复杂、需要深度 OA+财务+CRM 全链路协同，那可以考虑功能更重的 ERP 系统（如 jshERP 等）。
