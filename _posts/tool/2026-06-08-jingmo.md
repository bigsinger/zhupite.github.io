## 项目简介

京墨是一款开源的中华文化阅读 Android App，致力于将中华传统文化精华汇聚于移动端。从诗词歌赋到成语歇后语，从传统节日到二十四节气，从传统色谱到世界遗产，京墨以近乎百科全书式的数据覆盖，为中华文化爱好者提供了一个纯粹、无广告的阅读与查询工具。

项目采用 Material You 设计语言，界面清爽现代，数据全部离线可用。

## 使用方式 / 安装

京墨通过 F-Droid 和 GitHub Releases 分发，无需 Google Play 商店：

1. **从 GitHub Releases 下载 APK**
   - 访问 [Releases 页面](https://github.com/hefengbao/jingmo/releases)
   - 下载最新版本的 APK 文件
   - 在 Android 设备上安装（可能需要启用『未知来源』）
2. **从 F-Droid 安装**
   - 在 F-Droid 中搜索『京墨』
   - 点击安装

安装后无需联网，所有数据离线可用。

## 设计理念

京墨的设计理念围绕『离线可用』和『纯粹阅读』展开。所有数据打包在 App 中，无需网络即可访问全部内容，也没有任何广告或追踪。Material You 动态取色让界面能随系统主题变化，提供沉浸式的阅读体验。

从产品视角看，京墨不是简单的内容搬运工具，而是一个试图为中华传统文化在移动端建立『数字档案』的项目。其 160 万篇诗文的体量，以及从汉字、词语到对联、绕口令的全面覆盖，已经超越了绝大多数商业竞品。

## 社区与生态

京墨的数据源来自多个知名开源中文数据项目，包括 chinese-poetry（古诗词数据库）、hanzi-writer-data（汉字笔画数据）等。这种『站在巨人肩膀上』的整合方式，使得京墨能以较小的维护成本提供海量内容。

作为 GPL-3.0 许可的开源项目，京墨欢迎社区贡献。如果你发现遗漏的文化内容或需要新增数据类型，可以通过 GitHub Issues 或 Pull Request 参与项目共建。
