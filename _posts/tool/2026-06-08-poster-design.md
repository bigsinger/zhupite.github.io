## 迅排设计 (Poster Design) 是什么

**迅排设计（Poster Design）** 是一款美观且功能强大的在线图片编辑器和海报设计工具，仿稿定设计等产品。它基于 Vue3 + Canvas 构建，支持原生 DOM 画布编辑器和拖拽操作，提供了丰富的交互细节和基础功能。适用于海报图片生成、电商分享图、文章长图、视频/公众号封面等多种场景。

项目在 GitHub 上已获得 **4,715** 颗星，是在线设计领域受欢迎的开源项目。

> 迅排设计 - 美观且功能强大的图片编辑器、在线海报设计，让设计如此简单！

## 快速上手

**本地运行：**
```bash
git clone https://github.com/palxiao/poster-design.git
cd poster-design
npm run prepared
npm run serve
```

访问 `http://127.0.0.1:5173/` 即可开始使用。

**在线体验：**
访问 [迅排设计官网](https://design.palxp.cn/) 即可在线使用，无需安装。

**技术栈：**
- 前端：Vue3 + Vite5 + Pinia + ElementPlus
- 后端：Express + Puppeteer
- 图片生成：Puppeteer、Html2canvas
- 其他：Moveable（拖拽）、PSD 解析器

## 优劣势

**优势：**
- 完全开源，可自行部署，数据安全可控
- 交互体验流畅，接近原生桌面应用
- 支持 PSD 导入，兼容 Photoshop 设计稿
- AI 抠图功能实用，一键去除背景
- 前端 + 服务端混合出图，图像质量高
- 技术栈现代化，扩展性好

**劣势：**
- 功能丰富度相比稿定设计等商业产品仍有差距
- 在线运行时依赖后端服务进行图片渲染
- 大文件 PSD 导入解析可能较慢
- UI 部分细节仍有优化空间
- 社区插件生态尚未建立
