## DevToys 是什么

**DevToys** 是一款面向开发者的桌面工具集，堪称开发者的"瑞士军刀"。它内置了 30 多种常用开发工具，涵盖格式转换、编解码、格式化、生成器、图形工具等多个类别。DevToys 最棒的特性是智能检测功能——复制数据到剪贴板后，它会自动匹配最佳工具进行处理，无需手动选择。所有工具离线可用，无需联网，保护数据隐私。

项目在 GitHub 上已获得 **31,607** 颗星，是目前最受欢迎的开源开发者工具集之一。

> A Swiss Army knife for developers.

## 核心特性

**DevToys 2.0 内置 30+ 工具，覆盖七大类别：**

**转换器：**
- JSON <> YAML 互相转换
- 日期格式转换（时间戳、ISO 等）
- 数字进制转换（二进制、八进制、十进制、十六进制）
- 货币转换

**编码/解码：**
- HTML 编码/解码
- URL 编码/解码
- Base64 文本/图片编解码
- GZip 压缩/解压缩
- JWT 解码器
- 二维码生成器

**格式化：**
- JSON 格式化
- SQL 格式化
- XML 格式化

**生成器：**
- 哈希和校验和生成器（MD5, SHA1, SHA256 等）
- Lorem Ipsum 占位文本生成器
- 密码生成器
- UUID 生成器

**图形工具：**
- 色盲模拟器
- PNG/JPEG 压缩器
- 颜色选择与转换

**测试工具：**
- JSONPath 测试器
- 正则表达式（RegEx）测试器
- XML 测试器

**文本工具：**
- Markdown 预览
- 文本比较器（Diff）
- 文本分析与工具集

## 快速上手

**下载安装：**

- **Windows:** Microsoft Store 搜索 DevToys 或下载安装包
- **macOS:** 通过 Homebrew 安装 `brew install --cask devtoys`
- **Linux:** 通过 Snap 安装 `snap install devtoys`
- 也支持从 [GitHub Releases](https://github.com/DevToys-app/DevToys/releases) 直接下载

**扩展支持：**
DevToys 支持通过扩展插件添加更多工具，你也可以自行开发扩展。详见 [DevToys 开发文档](https://devtoys.app/doc)。

## 优劣势

**优势：**
- 30+ 工具一站式集成，无需在多个网站间切换
- 智能检测功能，自动匹配工具
- 完全离线可用，保护数据隐私
- 跨平台（Windows/macOS/Linux）
- 基于 WinUI 3 / Fluent Design，界面现代美观
- 开源免费，支持扩展

**劣势：**
- 部分工具的深度定制选项有限
- 某些场景下仍需专业独立工具
- 扩展生态仍在发展初期
- 部分高级功能不如专用工具全面

## 适合谁用

- 日常需要各种开发和调试工具的前后端开发者
- 经常处理 JSON/XML、编码转换的后端工程师
- 需要快速测试正则、Hash 的运维人员
- 希望在离线环境中工作的开发者
- 追求一站式工具平台的高效程序员
