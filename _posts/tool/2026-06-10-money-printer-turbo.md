# MoneyPrinterTurbo：AI 一键生成高清短视频

> 短视频创作不再需要专业的剪辑技能、昂贵的设备和漫长的制作周期——**AI 可以一键生成**。

**MoneyPrinterTurbo** 是由 **harry0703** 开发的一款 AI 驱动的短视频生成工具，利用大语言模型（LLM）以极简操作生成高清短视频。该项目在 GitHub 上获得了大量关注，成为 AI 内容创作领域的标志性开源项目之一。

## 核心能力

### 一键生成

用户只需输入一个主题或关键词，MoneyPrinterTurbo 就能自动完成：
1. **脚本撰写**：AI 基于主题生成视频脚本
2. **语音合成**：自动将脚本转换为语音旁白
3. **画面匹配**：搜索并匹配相应的视频画面或图片
4. **字幕生成**：自动识别语音并生成字幕
5. **视频合成**：将所有元素合成为完整的高清视频

### 支持多种输出风格

- 知识科普类
- 产品介绍类
- 新闻快讯类
- 故事叙述类
- 营销推广类

### 可定制参数

用户可以根据需求调整视频的：
- 时长（15秒～5分钟）
- 分辨率（720p、1080p、4K）
- 配音风格（多种语言和声线）
- 背景音乐
- 字幕样式

## 技术架构

```
输入（主题/关键词）
    ↓
AI 大模型 → 脚本生成
    ↓
TTS 引擎 → 语音合成
    ↓
媒体库/搜索引擎 → 画面匹配
    ↓
视频渲染引擎 → 最终输出
```

```bash

git clone https://github.com/harry0703/MoneyPrinterTurbo.git
cd MoneyPrinterTurbo

# 安装依赖
pip install -r requirements.txt

# 启动应用
python app.py
```
