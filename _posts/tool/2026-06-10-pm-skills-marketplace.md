# PM Skills Marketplace：用 AI Agent 做产品管理——68 个技能 + 42 个工作流

> 当 AI 被广泛应用于编码之后，下一个被赋能的角色是——**产品经理**。

**PM Skills Marketplace** 是一个革命性的开源项目，它将完整的产品管理方法论编码为 AI Agent 可执行的技能。如果你曾经为 Claude Code、Cowork 或其他 AI 助手感到"不够懂产品管理"而苦恼，这个项目就是你要的答案。

## 覆盖全链路：从发现到增长的 68 个技能

PM Skills Marketplace 提供了 **68 个 PM 技能** 和 **42 个链式工作流**，分布在 **9 个插件** 中，覆盖产品管理的完整生命周期：

### 发现阶段
| 命令 | 功能 |
|------|------|
| `/discover` | 启动需求发现流程 |
| `/user-research` | 组织用户研究方法 |
| `/competitive-analysis` | 竞品分析 |

### 策略阶段
| 命令 | 功能 |
|------|------|
| `/strategy` | 制定产品策略 |
| `/north-star` | 定义北极星指标 |
| `/okrs` | 设定 OKR 目标 |

### 执行阶段
| 命令 | 功能 |
|------|------|
| `/write-prd` | 撰写产品需求文档 |
| `/prioritize` | 优先级排序 |
| `/story-map` | 用户故事地图 |

### 发布与增长
| 命令 | 功能 |
|------|------|
| `/plan-launch` | 制定发布计划 |
| `/go-to-market` | 市场进入策略 |
| `/growth` | 增长实验设计 |

## 42 个链式工作流：从想法到执行

这些技能不是孤立的——你可以通过链式工作流将多个技能串联起来，形成完整的产品管理流程。例如：

- **新产品构思工作流**：`/discover` → `/user-research` → `/competitive-analysis` → `/strategy` → `/north-star` → `/write-prd`
- **发布检查工作流**：`/plan-launch` → `/go-to-market` → `/growth`
- **迭代优化工作流**：`/analyze-metrics` → `/prioritize` → `/story-map` → `/write-prd`

## 9 个插件：按需加载

工作流分为 9 个插件，用户可以根据自己的角色和需求选择性加载：

1. **Discovery** — 需求发现与用户研究
2. **Strategy** — 策略制定与目标设定
3. **Requirement** — 需求文档与规格
4. **Design** — 产品设计协作
5. **Execution** — 执行与交付管理
6. **Quality** — 质量保障
7. **Launch** — 产品发布
8. **Growth** — 增长与优化
9. **Analytics** — 数据分析与洞察

## 使用场景

- **独立产品经理**：需要快速产出 PRD 和策略文档
- **初创团队**：缺乏成熟的产品管理流程，需要方法论指导
- **咨询顾问**：为不同客户提供标准化的产品管理服务
- **AI Agent 开发者**：为自己的 Agent 添加专业的产品管理能力

## 结语

PM Skills Marketplace 让产品管理不再是"纸上谈兵"。它将行业最佳实践封装为 AI 可执行的技能模块，让每一个使用者都能像资深产品经理一样思考和行动。
