## Lantern 是什么

**Lantern**（蓝灯）是一款开源的 VPN 加速工具，致力于为用户提供高速的网络连接、隐私保护和网络审查突破能力。它是运营时间最长的开源 VPN 项目之一（始于 2011 年），支持 Android、iOS、Windows、macOS 和 Linux 全平台，免费即可使用。

Lantern 的核心工作原理是通过其分布式的全球代理网络，智能选择和路由用户的网络流量，实现快速、安全的网络访问。项目在 GitHub 上获得 **15,662** 颗星，拥有庞大的全球用户基础。

> Open-source VPN for speed, privacy, and censorship circumvention.

## 快速上手

**下载安装：**

根据您的操作系统选择对应版本：

**Windows 10+：**
- 稳定版: <https://github.com/getlantern/lantern/releases/latest/download/lantern-installer.exe>
- 测试版: <https://s3.amazonaws.com/lantern.io/releases/beta/latest/lantern-installer-beta.exe>

**Android：** 从 Google Play 或 GitHub Releases 下载 APK

**iOS：** 从 App Store 搜索 Lantern 下载

**macOS / Linux：** 从 GitHub Releases 页面获取对应安装包

**使用说明：**
1. 下载并安装对应平台的客户端
2. 打开 Lantern，无需注册即可使用免费版
3. 连接成功后，所有网络流量将自动通过加密通道传输

## 优劣势

**优势：**
- 运营超过 15 年，稳定性和可靠性经过时间检验
- 全平台覆盖，手机电脑都能用
- 开源透明，代码可审计
- 免费版即可满足基础使用需求
- 分布式架构，连接速度较稳定
- 无需复杂配置，下载即用

**劣势：**
- 免费版速度和流量可能受限
- Pro 版本需要付费订阅
- 部分地区可能无法正常连接
- 受网络环境影响，速度波动较大
- 用于突破审查的功能在部分国家/地区可能涉及法律风险

- 经常出国需要访问国内网站的用户
- 对外交流频繁的研究人员和跨国工作者
