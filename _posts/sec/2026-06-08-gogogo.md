## 概述

[GoGoGo（影梭）](https://github.com/ZCShou/GoGoGo) 是一款基于 Android 调试 API + 百度地图及定位 SDK 实现的虚拟定位修改工具，支持 Android 8.0+ 设备，**无需 ROOT 权限**即可随意修改当前位置以及模拟移动。

- **GitHub Stars**: ⭐ 10,593
- **语言**: Java
- **许可证**: GPL-3.0
- **Fork 数**: 1,207
- **最近更新**: 2025年11月

## 核心特性

### 1. 虚拟定位

利用 Android 的调试 API 实现位置模拟，支持：

- 精确设置经纬度坐标
- 在地图上直接选择位置
- 搜索地点快速定位
- 保存常用位置

### 2. 摇杆模拟移动

影梭的一大特色是**自由摇杆控制**，可以模拟真实移动：

- 支持摇杆方向控制
- 可调节移动速度（步行/跑步/驾车）
- 路线规划与自动行走
- 实时位置更新

### 3. 无需 ROOT

这是影梭最大的亮点之一——通过 Android 调试 API 实现，普通用户无需 ROOT 即可使用：

```bash
# 通过 ADB 授权（第一次使用）
adb shell settings put global mock_location 1
```

## 技术实现

影梭的核心技术基于：

1. **Android 调试 API** (`android.location.LocationManager.addTestProvider`)
2. **百度地图 SDK**：提供地图显示和定位接口
3. **百度定位 SDK**：辅助位置信息处理
4. **服务端模拟**：后台 Service 持续提供模拟位置
