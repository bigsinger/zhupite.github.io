# 逻辑渡河题大全（旗舰版）

## 1. 两警两贼

### 题目

两位警察和两位小偷要过河，只有一艘小船，每次最多载两人。小偷不能与警察单独留在同一岸边，否则小偷会被逮捕。小偷和警察都能划船，船上必须有人才能划。

### 解法步骤

1. 两个小偷过河（G G →），一个小偷返回（← G）
2. 两个警察过河（P P →），一个警察返回（← P）
3. 两个小偷过河（G G →），一个小偷返回（← G）
4. 两个小偷过河（G G →），全部抵达对岸

## 2. 三警三贼

### 题目

三个警察押送三个小偷过河，船每次最多运两个人。若任意一边小偷人数 > 警察人数，警察就会被小偷杀掉。

### 解法步骤

1. 两个小偷过河（G G →），一个小偷返回（← G）
2. 两个小偷过河（G G →），一个小偷返回（← G）
3. 两个警察过河（P P →），一个警察+一个小偷返回（← P G）
4. 两个警察过河（P P →），一个小偷返回（← G）
5. 两个小偷过河（G G →），一个小偷返回（← G）
6. 最后两个小偷过河（G G →），全员安全

## 3. 猎人、狼、羊与白菜

### 题目

猎人带狼、羊、白菜渡河，每次只能带猎人和其中一项。猎人不在时，狼会吃羊，羊会吃白菜。

### 解法步骤

1. 猎人带羊过河，猎人返回
2. 猎人带狼过河，放下狼带羊返回
3. 猎人带白菜过河，猎人返回
4. 猎人带羊过河，全员安全

## 4. 猎人、猎狗与两只绵羊

### 题目

### 解法步骤

1. 猎人带猎狗过河，返回
2. 猎人带羊过河，放下羊带猎狗返回
3. 猎人带另一只羊过河，返回
4. 猎人带猎狗过河，全员安全

### 题目

2. 两鬼过河（G G →），一鬼返回（← G）
3. 两人过河（P P →），一人+一鬼返回（← P G）
4. 两人过河（P P →），一鬼返回（← G）
5. 两鬼过河（G G →），一鬼返回（← G）
6. 最后两鬼过河（G G →），全员安全

## 8. 三只大老虎与三只小老虎

### 题目

大老虎 A,B,C 和小老虎 a,b,c 过河，船最多 2 只老虎，A,B,C和a会划船。小老虎不能离开母虎，否则被其他大老虎吃。

### 解法步骤

1. a和b过河（a返回）
2. a和c过河（a返回）
3. A和B过河（B和b返回）
4. C和A过河（a返回）
5. a和b过河（a返回）
6. a和c过河，全员安全
