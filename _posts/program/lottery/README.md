# 抽奖小程序使用说明

## 简介

这是一个简单易用的抽奖网页程序，使用 GitHub Pages 部署，支持用户自定义配置，使用本地存储技术保存配置。

## 功能特点

- ✅ 转盘抽奖，视觉效果好
- ✅ 支持自定义配置奖项
- ✅ 支持黑白名单过滤
- ✅ 支持中奖概率配置
- ✅ 预制图标，支持颜色背景
- ✅ 本地存储，配置永久有效
- ✅ 响应式设计，支持移动端
- ✅ 纯 HTML/CSS/JavaScript，无复杂框架

## 使用方法

### 1. 访问抽奖页面

在浏览器中打开抽奖页面 URL，即可开始使用。

### 2. 开始抽奖

点击"开始抽奖"按钮，转盘会开始旋转，最终停在中奖奖项上。

### 3. 配置设置

点击"设置"按钮，打开设置面板，可以配置以下内容：

#### 3.1 标题设置

在"标题设置"区域，输入您想要的标题，点击"保存"按钮即可。

#### 3.2 奖项配置

在"奖项配置"区域，可以：

- **添加奖项**：点击"添加奖项"按钮，添加新的奖项
- **删除奖项**：点击奖项右侧的"删除"按钮，删除该奖项
- **修改奖项**：直接修改奖项的名称、颜色和权重

**奖项说明**：
- **名称**：奖项的显示名称
- **颜色**：奖项在转盘上的背景颜色
- **权重**：奖项的中奖概率权重，权重越大，中奖概率越高

#### 3.3 黑白名单

在"黑白名单"区域，可以：

- **启用白名单**：勾选"启用白名单"，只有白名单中的奖项会参与抽奖
- **启用黑名单**：勾选"启用黑名单"，黑名单中的奖项不会参与抽奖
- **配置名单**：在文本框中输入奖项名称，每行一个

**示例**：

白名单：
```
一等奖
二等奖
三等奖
```

黑名单：
```
谢谢参与
```

#### 3.4 保存配置

配置完成后，点击"保存"按钮，配置会自动保存到本地存储中。

### 4. 查看使用说明

点击"使用说明"按钮，可以查看详细的使用说明。

## 配置说明

### 默认配置

程序启动时会加载默认配置，包含以下奖项：

1. 一等奖（权重：1）
2. 二等奖（权重：2）
3. 三等奖（权重：3）
4. 四等奖（权重：4）
5. 五等奖（权重：5）
6. 谢谢参与（权重：6）

### 权重说明

权重表示奖项的中奖概率，权重越大，中奖概率越高。

**示例**：

- 一等奖权重：1
- 二等奖权重：2
- 三等奖权重：3

则中奖概率为：
- 一等奖：1/6 ≈ 16.67%
- 二等奖：2/6 ≈ 33.33%
- 三等奖：3/6 = 50%

### 黑白优先级

黑白名单的优先级如下：

1. 如果启用了白名单，则只考虑白名单中的奖项
2. 如果启用了黑名单，则排除黑名单中的奖项
3. 如果同时启用白名单和黑名单，则先应用白名单，再应用黑名单

## 常见问题

### Q: 配置会丢失吗？

A: 不会，所有配置都保存在浏览器的本地存储中，永久有效。只要不清除浏览器缓存，配置就不会丢失。

### Q: 如何重置配置？

A: 在设置面板中删除所有奖项，然后保存即可。或者清除浏览器的本地存储。

### Q: 支持哪些浏览器？

A: 支持所有现代浏览器，包括：
- Chrome
- Firefox
- Safari
- Edge
- Opera

### Q: 可以在手机上使用吗？

A: 可以，程序支持响应式设计，可以在手机、平板、电脑等设备上使用。

### Q: 如何添加自定义图标？

A: 目前程序支持颜色背景和预制图标。如需添加自定义图标，可以修改代码中的图标路径。

### Q: 抽奖结果可以导出吗？

A: 目前版本不支持导出抽奖结果，但会在界面上显示中奖结果。

### Q: 可以设置抽奖次数限制吗？

A: 目前版本不支持抽奖次数限制，可以无限次抽奖。
