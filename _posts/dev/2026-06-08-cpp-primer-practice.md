# C++ Primer 练习 — 搞定 C++ 经典学习仓库

## 工具简介

**Cpp_Primer_Practice**（[GitHub](https://github.com/applenob/Cpp_Primer_Practice)）是由 applenob 发起的开源学习项目，使用 **C++** 语言编写，当前拥有 **8705** 颗星。它是《C++ Primer 中文版（第 5 版）》的配套学习仓库，包含完整的**学习笔记**和**课后练习答案**。

## 核心特性

- **章节全覆盖**：涵盖原书全部 19 章内容
- **笔记+练习**：每章包含知识点笔记和配套习题答案
- **多贡献者协作**：由多位开发者共同维护完善
- **环境友好**：基于 Ubuntu 16.04 + VS Code + g++

## 内容结构

- **第 I 部分**：C++ 基础（变量、字符串、表达式、语句、函数、类）
- **第 II 部分**：C++ 标准库（IO、顺序容器、泛型算法、关联容器、动态内存）
- **第 III 部分**：类设计工具（拷贝控制、重载、面向对象、模板）
- **第 IV 部分**：高级主题（特殊设施、大型程序工具、特殊工具与技术）

## 适用人群

- C++ 初学者系统学习
- 备考 C++ 相关认证
- 需要快速查阅 C++ 知识点的开发者
- 教学辅助材料
